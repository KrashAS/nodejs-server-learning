# ➗ Урок 6. Міні-API: Калькулятор

## Ціль:

-   створити `/calc` маршрут
-   обробляти параметри `a`, `b`, `op` (операція)
-   підтримка `GET` і `POST`
-   повернути результат у JSON

---

## Підтримувані операції:

| Операція | Значення |
| -------- | -------- |
| `add`    | a + b    |
| `sub`    | a - b    |
| `mul`    | a \* b   |
| `div`    | a / b    |

---

## Приклад `GET` запиту:

-   GET http://localhost:3000/calc?a=10&b=2&op=mul
-   Відповідь:

```json
{ "result": 20 }
```
